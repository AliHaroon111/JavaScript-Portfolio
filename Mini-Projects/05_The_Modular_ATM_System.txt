Mini Project: "The Modular ATM System"
Refactor (rewrite) a simple ATM logic using Functions.

Requirements:

Global Variable: let balance = 1000;
Function 1: checkBalance()
Prints: "Your Current Balance is: [balance]"
Function 2: deposit(amount)
Add amount to balance.
Print: "Deposited [amount]. New Balance: [balance]"
Function 3: withdraw(amount)
Check if amount > balance.

If yes, print "Insufficient Funds".

If no, subtract amount and print "Withdrawn [amount]. Remaining: [balance]".
